A Kotlin implementation of the [TCN](https://github.com/TCNCoalition/TCN) protocol.

## Installation

1. Add maven repository `https://dl.bintray.com/untori/untori`
2. Add TKN dependence `it.unisannio.assd.untori:{version}`
